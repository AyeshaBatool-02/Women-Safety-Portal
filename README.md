# 🛡️ Women Safety Portal

## Empowering Safety Through Technology

Women Safety Portal is a web-based application developed to provide a secure and accessible platform for women to report incidents, manage safety-related information, and access location-based assistance features.

The project focuses on improving personal safety by offering a centralized portal where users can register, log in, report incidents, manage profiles, and access safety resources efficiently.

---

# 📌 Project Overview

Women's safety remains an important social concern. This project was developed as a database-driven web application to help users report incidents and access essential safety-related services through a user-friendly interface.

The system combines authentication, incident management, profile management, and location tracking features into a single platform.

# 🚀 Installation Guide

## Step 1: Clone Repository

```bash
git clone https://github.com/yourusername/women-safety-portal.git
```

## Step 2: Move Project

Copy project folder to:

```text
xampp/htdocs/
```

## Step 3: Start Services

Open XAMPP and start:

* Apache
* MySQL

## Step 4: Create Database

1. Open phpMyAdmin
2. Create a new database
3. Import:

```text
create-database.sql
```

## Step 5: Run Application

Open browser:

```text
http://localhost/project-folder-name
```
